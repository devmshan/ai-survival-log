## 초록 요약

기존 triple 기반 지식그래프(KG)는 시간 역학, 출처 세부사항 등 관계 지식의 맥락 정보가 부족하다. 이 논문은 Context Graphs(CG)를 제안하고, LLM을 활용한 CGR³ 패러다임(Retrieve-Rank-Reason)을 소개한다.

## 주요 기여

1. 삼중항 기반 KG의 한계 분석 및 CG 개념 도입
2. LLM 기반 CGR³ 패러다임 제안
3. KGC, KGQA 두 작업에서 성능 향상 증명

## CGR³ 방법론 (Retrieve → Rank → Reason)

### 1. Retrieve
- Supporting Triple Retrieval
- Textual Context Retrieval (Wikidata + Wikipedia)
- Candidate from KG embedding
- Candidate from LLM 생성

### 2. Rank
- SFT + LoRA 기반 Llama3-8B
- 질문-후보-맥락 정합성 평가

### 3. Reason
- 반복적 추론 (충분한 정보 확보까지)
- GPT-3.5-turbo-0125 활용

## 실험 데이터셋

### KGC
- FB15k-237: 14,541 엔티티, 237 관계, 272,115 학습 삼중항
- YAGO3-10: 123,182 엔티티, 37 관계, 1,079,040 학습 삼중항

### KGQA
- QALD10-en (Wikidata 기반)
- WikiWebQuestion/WWQ

## 결과

| 작업 | 지표 | CGR³ | SOTA 대비 |
|------|------|------|-----------|
| KGC (FB15k-237) | Hits@1 | avg +33% | vs ComplEx/RotatE/GIE |
| KGQA (QALD10-en) | Acc | 54.7% | +43.6% vs prior |
| KGQA (WWQ) | Acc | 78.8% | +17.1% vs prior |

- GPU: NVIDIA A100-SXM4-40GB × 8
