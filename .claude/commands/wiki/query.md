# /wiki:query — 위키 검색 + 인용 답변

위키 전체를 검색하여 질문에 답변하고, 출처를 인용합니다.

## 입력

사용자의 질문. 인자가 없으면 질문을 요청합니다.

## 워크플로우

### 1단계: 인덱스 검색

- `wiki/index.md`를 읽어 관련 페이지를 식별합니다
- 질문의 키워드, 태그, 카테고리를 기준으로 매칭합니다
- 필요하면 `wiki/sources/`를 우선 보고, 원본이 더 필요할 때만 대응하는 `raw/{type}/` 파일까지 확인합니다

### 2단계: 관련 페이지 읽기

- 식별된 관련 페이지를 모두 읽습니다
- 필요 시 해당 페이지의 `## 관련 페이지`를 따라가며 추가 페이지도 읽습니다
- 비교, 통합, 판단 성격의 질문이면 `wiki/syntheses/` 후보도 함께 확인합니다

### 3단계: 답변 생성

- `[[wikilink]]` 인용과 함께 답변을 작성합니다
- 답변 형식:

```markdown
## 답변

[답변 내용, [[entities/claude-code]] 등 인용 포함]

### 참조한 페이지
- [[entities/claude-code]] — Claude Code 엔티티
- [[concepts/ax-ai-transformation]] — AX 개념
```

### 4단계: 저장 제안

- 답변이 새로운 지식을 포함하면: "이 답변을 위키 페이지로 저장하시겠습니까? `/wiki:file-answer`로 저장할 수 있습니다."

## 주의사항

- 위키에 없는 정보는 "위키에 해당 정보가 없습니다"라고 명시
- 추측하지 않고, 위키에 있는 정보만 인용
- 위키 외부 지식이 필요하면 `/wiki:ingest`로 소스 추가를 제안
- 웹 자료를 새로 받아야 하면 `assets/clipper-templates/`의 적절한 템플릿으로 `raw/{type}/` 또는 `assets/intake/`에 먼저 저장하도록 안내합니다
