# Ralph Loop (랄프 루프)

"구현 → 검토 → 개선"을 자기참조적으로 반복하는 루프. AI가 이전 결과를 스스로 평가하고 다음 방향을 결정하는 방식. 한 번에 완성되지 않는다 — 반복이 다듬는다.

## 개념

```
구현(Implement)
    ↓
검토(Review) — AI가 이전 결과를 스스로 평가
    ↓
개선(Improve)
    ↓ (자기참조적 반복)
```

## 핵심 특성

- **자기참조적(Self-referential):** 루프 자체가 자신의 출력을 입력으로 사용
- **반복 중심:** 한 번에 완성을 목표로 하지 않음. 반복이 품질을 만든다
- **AI 평가 포함:** AI가 단순 실행자가 아닌 평가자 역할도 수행

## diversification → selection → amplification

Ralph Loop와 함께 쓰이는 자기 개선 패턴:

| 단계 | 설명 | 비유 |
|------|------|------|
| **Diversification** | 현재 상태를 모두 꺼내놓기 | 코드 모듈 분해 |
| **Selection** | 흐릿한 것은 버리고, 진한 것만 남기기 | 불필요한 의존성 제거 |
| **Amplification** | 선명하게 남긴 것을 강화 | 핵심 기능 최적화 |

산업의 bundle → unbundle 사이클과 동일한 구조를 개인 성장에 적용.

## AI 개선 방식의 인간 적용

> *"AI가 코드를 개선하는 방식으로, 나 자신을 개선한다."*

[[concepts/rlvr|RLVR]]이 AI를 학습시키는 방식 — 결과를 평가하고 더 나은 방향으로 반복 — 을 자기 성장에 메타포로 적용.
