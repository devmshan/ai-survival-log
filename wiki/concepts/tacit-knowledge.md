# 암묵지 (Tacit Knowledge)

Michael Polanyi의 개념: **"We know more than we can tell"** — 말로 표현하기 어려운 앎. AI 시대에 상대적 가치가 부각되는 인간 고유의 지식 형태.

## 정의

암묵지는 경험과 실천을 통해 체화된 지식으로, 명시적 규칙이나 문서로 완전히 표현되기 어렵다. 명시지(Explicit Knowledge)와 대비된다.

```
명시지(Explicit) → 문서화됨 → AI가 학습 가능 → 빠른 자동화
암묵지(Tacit)   → 체화됨   → 데이터 없음   → 자동화 어려움
```

## AI 시대에 암묵지라 불리는 것들

### 1. 경험적 직관

- 수년간 쌓인 "이상하다"는 감각
- 장인이 재료를 만져보고 아는 것
- 임상의사의 "이 환자 뭔가 이상하다" — 검사 수치 이전의 직관

### 2. 맥락적 판단

- 규칙은 알지만 언제 깨야 하는지 아는 것
- 회의실 분위기를 읽고 타이밍을 잡는 것
- 고객이 말한 것 vs 원하는 것의 차이를 아는 것

### 3. 관계적·사회적 지식

- 이 사람을 어떻게 설득해야 하는지
- 팀의 신뢰를 쌓는 방식
- 조직 내 권력 구조를 읽는 능력

### 4. 미적·창의적 판단

- "이 디자인이 왜 안 좋은지" 설명은 못 하지만 아는 것
- 글의 리듬과 어조
- 무엇을 버려야 하는지

## RLVR과의 관계

> "암묵적 신뢰, 맥락적 판단, 미적 영역은 아직 AI가 닿지 않는 곳" — [[topics/ai-era-survival]]

## AI 시대에 암묵지의 가치

**역설:** AI가 명시지를 빠르게 처리할수록, 상대적으로 암묵지의 가치가 올라간다.

- AI가 코딩, 법률 자문, 의료 진단의 명시적 부분을 처리
- 남는 것은 판단, 신뢰, 맥락 — 암묵지 영역
- "무엇을 만들지" 결정하는 능력 vs "어떻게 만들지" 실행하는 능력의 분리
