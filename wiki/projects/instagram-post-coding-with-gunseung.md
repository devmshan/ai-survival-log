### 슬라이드 4 — 수정 과정 (캡처본 스타일)
```
┌──────────────────────────────────────┐
│  배경: #0d1117                        │
│                                      │
│  > 직접 수정한 것들                   │
│  ─────────────────────────           │
│                                      │
│  # error: 건승이가 흰 고양이였다      │
│  # fix:   실제 포스트 URL 제공 → 수정 │
│                                      │
│  # error: 후드가 네이비였다           │
│  # fix:   회색으로 변경               │
│           (개발자처럼 보이고 싶었음)  │
│                                      │
│  # error: 배경이 어색했다             │
│  # fix:   집 사진 직접 첨부           │
│           → 흰 커튼 배경으로 반영     │
│                                      │
└──────────────────────────────────────┘
```
- `# error / # fix` 주석으로 수정 과정 담음
- 원하는 결과 = 직접 데이터를 넣어야 나온다는 흐름

## Canva 제작 가이드

**공통 설정**
- 사이즈: 1080 x 1080 (정사각형)
- 배경색: `#0d1117`
- 본문 폰트: Space Mono (Canva 내 검색)
- 본문 색: `#e6edf3`
- 강조 색: `#4fc3f7` (틸 블루)
- 주석(`#`) 색: `#8b949e` (회색)

**슬라이드 1**
- 이미지 전체 배치
- 좌하단 텍스트: Space Mono 12px, 흰색 60% 투명도

**슬라이드 2-4 (캡처본)**
- 좌측 상단 `>` 프롬프트 기호
- 구분선: 1px, `#30363d`
- 코드 블록 느낌으로 들여쓰기

**슬라이드 5**
- 배경 그라데이션: `#0d1b2a` → `#1a2744` (위→아래)
- 폰트: 얇은 고딕 (Noto Sans KR Light)
