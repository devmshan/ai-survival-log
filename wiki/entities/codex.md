# Codex

OpenAI의 코딩 에이전트. Claude Code와 같은 AI 코딩 도구 카테고리에 속한다.

2026년 4월 기준 OpenAI가 앱 형태로 출시했으며, Claude Code의 경쟁 제품으로 자리잡고 있다.

## 이 위키에서의 맥락

개인 Claude Code Max 구독자가 토큰 소모 체감 급증 문제로 인해 비교 검토한 대안 도구. 이 위키를 관리하는 사용자는 Claude Code와 Codex를 이중 스택으로 써본 경험을 바탕으로 역할 분담 구조를 설계했다.

- Claude Code: Planning/architecture에 강함
- Codex: 실행/검증 레이어로 보완

주관적 체감 비교 (2026-04 기준):
- Claude Code는 빠르지만 더 자주 수정이 필요하고 토큰 소모가 큼
- Codex는 같은 결과물 대비 토큰 소모가 적게 체감됨

공식 스펙보다 개인 사용 경험 중심의 요약이며, 추후 공식 문서 참조로 보강 가능.
