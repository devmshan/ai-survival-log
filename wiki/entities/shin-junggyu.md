# 신정규

## 개요

[[entities/lablup|Lablup]] 대표. 물리학과와 컴퓨터공학을 복수전공한 배경을 가진 개발자 출신 창업자. AI infrastructure 분야에서 11년간 Lablup을 이끌어 왔으며, [[concepts/agentic-workflow|Agentic Workflow]] 방법론의 실천가로서 대규모 AI 코딩 프로젝트를 직접 수행했다.

## 주요 이력

| 항목 | 내용 |
|------|------|
| 전공 | 물리학과 + 컴퓨터공학 복수전공 |
| 오픈소스 경험 | TextCube 블로그 플랫폼 프로젝트 참여 |
| 창업 | [[entities/lablup|Lablup]] 설립 및 대표 (2026년 기준 11년차) |
| 주요 성과 | [[entities/backend-ai|Backend.AI]]:GO를 40일 만에 Claude Code로 개발 |

## Agentic Workflow 실천 사례

2026년 초, Claude Code Max 플랜을 사용해 **40일 / 130억 토큰 / 100만 줄 코드**로 Backend.AI:GO를 완성했다. 단순히 AI 코딩 도구를 쓴 것이 아니라, harness 설계와 태스크 분해, 컨텍스트 관리 등 체계적인 Agentic Workflow를 적용한 결과다.

이 경험을 [[entities/ai-frontier|AI Frontier]] EP86에서 공개하며 다음과 같은 통찰을 공유했다:

- **Claude Code의 경쟁력은 harness:** 엔진(모델)보다 감싸는 방식이 핵심
- **사이버 포뮬러 비유:** Claude Code(아스라다) = 인간-AI 공진화, Codex(오가) = AI 단독 최적화
- **"내 일이 줄어드는 게 핵심":** Agentic Workflow의 목표는 AI에 루틴을 위임해 인간이 본질적 판단에 집중하는 것

## 철학

- **[[concepts/startup-watermill|물레방아론]]:** IT와 도메인의 교차점, 낙차 큰 곳에 스타트업 기회가 있다
- **[[concepts/bio-token|바이오 토큰]] 개념:** AI 시대 인간의 인지/에너지 자원을 효율적으로 배분해야 한다
- **[[concepts/code-value-convergence|코드 가치 0 수렴론]]:** 코드 자체의 가치는 0으로 수렴, 모델과 도메인 지식이 진짜 자산
