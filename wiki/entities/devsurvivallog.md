# devsurvivallog (dev.survival.log)

AI 시대에서 살아남으려고 애쓰는 개발자의 블로그. 화려한 성공담이 아닌, 매일의 시행착오와 배움의 기록. 2026-03-09에 시작.

## 이름의 의미

`dev.survival.log` — 개발자의 언어로 자기 자신을 정의한 이름.

| 키워드 | 의미 |
|--------|------|
| **dev** | 나라는 사람을 개발하다. 지금의 나는 초안이다. |
| **debugging** | 나태함도 버그. 회피하지 않고 해결하며 견고해지기 |
| **update** | AI 세상은 6개월이 다름. 멈추면 deprecated. 계속 달리기 |
| **optimize** | 나답게 최적화. 효율적으로 다듬되 내 향기는 유지 |

## 시작 배경

- **2026-03-09:** 스레드에서 Claude Code 얘기가 도배됨 → "뭔데 이렇게 난리야?"
- 경주는 이미 시작되어 있었고, 뒤늦게 합류
- [[concepts/ai-capability-overhang|AI Capability Overhang]] 상태: 도구는 있지만 활용 역량이 부족

## 철학

> "AI로 빠르게 수렴하고, 나의 언어로 발산하는 공간이다."

> "잘 살아온 기록이 아니라, 지금 살아가는 방식의 기록이다."

**[[concepts/ralph-loop|Ralph Loop]] 적용:** diversification → selection → amplification. 나라는 코드를 모듈로 분해하고, selection하고, amplification하여 재조립.
